Administración de la Aplicación

Esta sección explica cómo usar la ventana de administración (ventAdministracion) y las opciones del menú.

Acceso al menú

Desde la ventana de administración, usa el menú para acceder a las siguientes funcionalidades:

Gestión de eventos

  • Ver eventos: Selecciona "Gestión de eventos" para abrir ventEventos y revisar la lista de eventos, y editarlo como administrador.
  • Crear evento: Selecciona "Crear eventos" para abrir ventCrearEvento, llena los datos y guarda.
  • Actualizar evento: Selecciona "Ver >> eventos", elige un evento y pulsa "Actualizar" para editarlo.
  • Eliminar evento: Selecciona "Ver >> eventos", elige un usuario y pulsa "Eliminar" para eliminarlo.

Gestión de usuarios

  • Ver usuarios: Selecciona "Gestión de usuarios" para abrir ventUsuarios y revisar la lista, además puedes editarla como administrador.
  • Crear usuario: Selecciona "Crear personas" para abrir ventCrearUsuario, ingresa los datos y guarda.
  • Actualizar usuario: Selecciona "Ver >> usuarios", elige un usuario y pulsa "Actualizar" para editarlo.
  • Eliminar usuario: Selecciona "Ver >> usuarios", elige un usuario y pulsa "Eliminar" para eliminarlo.

Gestión de Informes

  • Crear informes: Selecciona "Crear >> Informes" para abrir la ventana ventInformes, en ella podras generar los 3 tipos de informes mostrados. Selecciona un formato y pulsa el botón del informe que deseas generar.

Uso ventAdministracion, botones y filtros

Asociar usuarios a eventos

  • Selecciona "Asociar" para abrir ventAsociar, elige un evento y un usuario, y haz clic en "Asociar".
  • También disponemos de un botón "Eliminar Asociacion" por si quieres eliminar la asociación entre un usuario y evento.

Panel de filtros

  • Aquí puedes filtrar para buscar resultados más rápido en la tabla con los campos mostrantes y pulsar el botón "Filtrar" o enter.

Panel de botones

  • Aquí puedes pasar de página para ver más resultados, y también vaciar los campos de los filtros.
