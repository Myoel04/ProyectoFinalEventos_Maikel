Interfaz de los Eventos de cada Usuario

Esta sección te ayudará a entender cómo utilizar la interfaz de búsqueda y visualización de eventos en la aplicación.

Componentes de la Interfaz

  • Título: Ingresa parte o todo el título del evento que deseas encontrar. Este campo es sensible a las coincidencias de texto introducido.
  • Tipo de Evento: Utiliza este menú desplegable para seleccionar un tipo de evento específico o 'Todos' para buscar eventos de diferentes tipos.
  • Botón Filtrar: Haz clic en este botón después de ajustar los filtros deseados para actualizar la lista de eventos mostrados en la tabla.
  • Tabla de Eventos: Aquí se muestran los eventos que tienes como usuario.
  • Botones de Navegación: Utiliza los botones 'Anterior' y 'Siguiente' para navegar entre páginas de resultados si hay más eventos que los que se pueden mostrar en una sola página.
  • Botón Vaciar: Haz clic en este botón para borrar los filtros aplicados y mostrar todos los eventos disponibles.

Uso Eficiente

Se recomienda utilizar los filtros de manera combinada para precisar la búsqueda según tus necesidades específicas. Esto te ayudará a encontrar los eventos más relevantes más rápidamente.
