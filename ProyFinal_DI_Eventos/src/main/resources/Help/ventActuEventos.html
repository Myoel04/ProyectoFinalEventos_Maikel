Actualizar Evento

Esta página permite modificar la información de un evento ya registrado en el sistema.

Instrucciones de Uso

  • Cómo podemos observar, están los campos rellenados con los datos actuales. Modifica el que veas necesario y pulsa el botón "Guardar Cambios".

Botones de Acción

Utiliza el botón "Guardar cambios" para aplicar las actualizaciones hechas al evento. Si decides no realizar cambios, puedes presionar el botón "Cancelar" para volver a la página anterior sin modificar nada.
